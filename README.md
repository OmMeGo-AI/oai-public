# OAI Public

Public open-source release from OmMeGo.AI Inc.

This repository is intended to contain:
- sub-agent templates
- recipe templates for sub-agent usage
- examples
- supporting documentation

## Repository layout

- `docs/` — paper, slides, and release documentation
- `examples/` — usage examples
- `agents/templates/` — reusable sub-agent templates
- `agents/recipes/` — reusable recipe templates
